Basics

Catch Data

These settings define how the fish behaves and when it is eligible to spawn. Exact places are handled in the Locations section below.

Location Library

Click a location to use it. If a location field is focused, the library fills that row. Otherwise it adds a new location row. Modded examples are included as helpers, but free text still works for anything not listed here.

Locations

These are the actual locations where the fish can spawn. Free text is allowed. For modded places like SVE, East Scarp, or RSV, enter the exact internal location name used by that mod.

Produced Items

Each card below is one possible pond output. Pick the item by name, then set the population threshold, chance, and quantity range. The export code is shown for reference only.

Fish ponds check these top-to-bottom and take the first valid entry that passes chance, so higher-population/special rewards should be above Roe. The editor now sorts them that way automatically.
