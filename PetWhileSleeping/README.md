# Pet While Sleeping

Pet While Sleeping lets you pet sleeping barn and coop animals at night so forgetting until late no longer costs you the daily friendship gain.

## Install

1. Install SMAPI.
2. Put this mod in your Stardew Valley `Mods` folder.
3. Launch the game.
4. Optionally toggle the mod in Generic Mod Config Menu.

## Notes

- The mod only changes the late-night sleep refusal in `FarmAnimal.pet(...)`.
- Normal petting behavior, friendship gain, and the animal info menu stay vanilla.
- Generic Mod Config Menu is optional.
